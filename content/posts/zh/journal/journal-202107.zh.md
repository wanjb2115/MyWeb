# 07/11

大苹启程回家，开启独自内卷时光。

# 07/10

结婚一周年纪念日，在经历财政变故之后，和大苹重建财务信任，一起为以后家庭资金制定发展计划 - Weebing基金。

下午去家具城逛了逛，国际家具城和生活馆基本买不起，只能感叹有钱真好。不过去了南京宜家，淘了
好多东西，地方是真的大，买的也都是十分精致的小物件，很难不买什么东西，又剁了一次手。

# 07/09
上午打卡南昌火车站，然后全力赶回家，中间途径合肥，打卡安徽省会，晚上7点到南京。

回公寓整顿了一下，发了个躺平朋友圈，也是早有设计，不过，旅游可以躺，人生还得闯！

# 07/08
上午准备离开长沙，昨天橘子洲景区卖的艺术雕像纪念品始终让我念念不忘，于是乎，上午果断去买了99的
车载艺术雕像，让长沙之旅不留遗憾，顺便打卡了汴京炸鸡，属实和正常炸鸡不同，特别特别嫩和香，大苹
都爱了。

中午启程回家，11个小时的路程属实太长，突然发现离江西省会南昌很近，一个小心思便产生了：和大苹打卡
所有的省会城市，刚走了1个小时便更改目的地到南昌，打卡江西省会。

晚上7点到南昌，停车也是不好停，酒店前车位紧张，动一动车都得打电话让别人调车。晚上大苹说带了一路的
泳衣没用上，所以临时起意去游泳馆游泳，37一人不算便宜，还好环境不错，跟大苹游了两小时，大苹进步明显，
逐步适应水的浮力，值得表扬。

# 07/07
昨晚青旅没睡好，一早7点便早早离开，打算把东西放到今晚酒店的停车场，长沙停车也不容易，得找个
免费停车的酒店，早上9点到了晚上要入住的酒店，运气很好刚好有个空房间打扫好了，提前办理入住，
进去休息了一会儿。

根据大苹的攻略，中午排号了么子烤肉，11.30店开始营业，11.35我们到店，没怎么排队直接开吃，
服务蛮周到的，类似海底捞，菜量也足足，看到美团上人均80,我们也照这标准点，最后点了150没吃完，
味道不错，菜量足，评分高，值得推荐。

吃过饭去小吃街逛了逛，打卡茶言悦色、黑色经典等系列小吃，味道还不错，不过店铺开的是真的密集，你可以
看到同一个路口有4个茶言悦色店铺，而且每个都在排队...真心的超网红产品。

下午去橘子洲，除了节假日可以把车开到景区，我们不了解，硬是自己骑车+步行走到了景区门口，到景区
一般就得坐观光车了，因为景区很大，走的话得5km才到雕像，景区没门票，观光车40/人可以上下多次，
价格也算可以了，感觉比庐山要实惠一些。橘子洲景区最标志性的便是毛主席青年艺术雕像了，来长沙也
是必打卡项目，到了看着果然很震撼，拍照的意愿十分强烈，看到雕像由衷感叹：风华正茂。

晚上去了IFS国金中心，7楼的人特别多，基本都是来打卡网红kaws雕像，排队打卡，我们去看了看也就那样。

# 07/06
大苹说庐山冷，上午去九江的服装市场买厚衣服，大苹逛起来一发不可收拾，说服装价格比南京
便宜多了，反季厚衣服没买着，给糖糖买了好几身衣服，我顺便去装了行车记录仪，就出发去庐山
景区了。

到了庐山景区门口才发现，在7.1-8.31暑假期间不允许车辆进入庐山，必须坐观光车，90/位只送到
庐山山上的门口，进山还得坐车，合计合计人均300都不一定能上山，果断拒绝了，订的山上的宾馆也
不支持取消，和商家大吵了一架最后商量退一半钱，下次一定得看好景区公告再制定旅游计划。

庐山不打算去了，转头就开往长沙，路上经过了庐山西海景区，是个新开发的景区，看着还不错就打算
下高速去逛一逛，庐山西海旁边建满了度假小区和酒店，不过目前还没啥人住，景区也不错，山上的一个
超大的湖泊，加上蓝天白云绿草，拍照效果nice，而且景区目前免费，还没建好，值得推荐。

晚八点到长沙，最大的冲击便是路上行人和电动车都不看红灯的，都是很自信的闯过去，红绿灯形同虚设，
或许这就是城市特色吧。一到酒店发现订错了，订的是明天的酒店，迅速找了个青年旅社住下，我住5人间，
有3个伙计凌晨2点进门就开灯，把我吵醒了，心情很郁闷。

# 07/05
篁岭之旅，早上在旅馆订的早餐，10元一碗煎蛋青菜面，味道还不错，吃完就出发了。

篁岭建在山顶上，与世隔绝，必须坐缆车过去，昨晚在旅店老板那里买了优惠票，篁岭120，加
10元送石门峡景区，算起来比美团划算，身份证验票，再也不用担心票丢了的问题了。

篁岭商业化是比较严重的，几乎所有房屋都在卖东西，原住村民或许也改行加入商人队伍中去，不过
景区特别大，水街、玻璃栈道、晒秋、冰雪世界、梯田、天街...能去的几乎都去了，而且景区还在扩建中，
未来应该会有更多景点。总的来说还不错吧，只是最重要的梯田油菜花没能看到，算是小小的遗憾。
130的门票包含缆车、玻璃栈道、水街、冰雪世界，也算可以接受了。

篁岭总体来说不错，篁岭石门峡景区若是原价买便有些感觉不值得，原价55的门票，由于我们的优惠票
送了石门峡景区，所以便决定去看一看。总的来说便是一道大峡谷，一条时而湍急，时而缓和的大溪流从
峡谷中穿出来，构成这个景区。人特别少，几乎没有人，主打的还是需要另外付费的英雄闯关，并且距离
景区特别远，在景区门口3km左右，逛完篁岭，身心疲惫，走到一半便回去了。路上也没有什么东西，人造
的景观居多，有一段可以戏水的区域，总的来说，免费的来玩玩水可以，原价真心不值。

晚上赶往庐山，在九江市威尼斯花园酒店住下，刚到酒店门口还遇到查酒驾，我们第一次被查酒驾，吹一下
就让通过了，还是蛮神奇的。

# 07/04
规划好了5号去篁岭，4号就随便逛一哈，去了漳村和月亮湾，都是免费的景点，漳村真的是算是
较为原始的山间村落，全村500余人，每家每户都有姓名和家训，村子里水牛在吃草，狗狗在追逐，
实实在在的乡村场景。大苹一开始还打着伞，看到整个村落也没人打伞，自己也不好意思再继续打了。

上午村落河面上起了雾气，好是神秘，悠然的雾气，蔚蓝的天空，错落的村屋，很有一番韵味。

下午来到篁岭民宿附近提前住下，老板是一家五口人，主要是爸爸妈妈在经营，爷爷奶奶打下手，
孩子在民宿玩耍，装修很简约，卫生很整洁，贵有贵的道理，非常不错的民宿，一家人也很友善。

晚上去楼下中堂吃饭时，大苹递给我一本《穷爸爸富爸爸》的书，我看了前几页就被内容所吸引，
接着便一发不可收拾，作者大思想跟我很像，小细节也讲的让我醍壶灌顶，受益匪浅，虽说之前
经历过破产风波，让整个家庭产生了恐惧心理。但核心思想仍然不能放弃，教育和投资才是值得
一辈子去付出的方向。

# 07/03
上午去搬最后一趟家，跟房东交房，遇到一个好房东真的很舒服，全程合作顺利，交接顺利。

下午收拾收拾决定出发去江西婺源了，上班前的最后一次休假，下午三点半出发，5个小时的路程，
中途下了好大的雨，晚上7.30左右还开车在山间高速驰骋。

黑天暴雨，一直开到晚上9.30才到婺源县城，十分难忘的经历。

# 07/02
昨天搬家还没搬完，打算今天最后一趟，可是房东有事，今日只能先鸽一哈。

上午平平淡淡，下午充分搞了搞大学入学时买的电脑，想把2015年的电脑改成Linux服务器。

装了一下午的Arch系统，也算磕磕绊绊装成功了，在这过程中我深深地感觉到Arch才是Linuxer的最终归宿，
用ubuntu和centOS也就图一乐，真正的Linux系统还得看Arch。

大苹整了一天的家，有个贤惠的妻子真的很幸福。

# 07/01
7月的第一天，下半年正式开始，但开局便是挑战——搬家。

7.1日全天都在搬家，上午叫了一辆货拉拉，加上自己的小polo搬了一趟，不得不说，东西属实有点多，从4楼
搬下，再搬上11楼，来来回回好是辛苦。

新公寓的小家一下子就被装满了，从88平下降到50平，东西却没减少多少，给房间空间布局带来了很大压力。

由于公寓的床也只有1.4米宽，根本不够两个人睡，所以我们又修改了房间的布局，将两个1.4米的床拼起来得
到3米的超级大床，整个房间也瞬间被床占据。

其余还有好多东西没有收拾，但是我们拥有了睡觉空间的保障，牺牲80%的房间空间得到3米大床充分满足睡眠需求。
